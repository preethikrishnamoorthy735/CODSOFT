## Description
This project is part of the **CodSoft Tech Internship**.  
It is a simple **Rock Paper Scissors game** developed using Python, where the user plays against the computer.

The computer randomly selects an option, and the winner is decided based on standard game rules.

## Features
- User input: rock, paper, or scissors
- Computer makes a random choice
- Displays win, lose, or tie result
- Score tracking for user and computer
- Option to play again

## How to Run
1. Make sure Python is installed on your system.
2. Open VS Code and open the **TASK-4** folder.
3. Open the integrated terminal in VS Code.
4. Run the program using:python rock_paper_scissors.py
5. Enter your choice when prompted and play the game.
